# 📊 Customer Behaviour Analysis

## 📌 Overview
This project focuses on analyzing customer shopping behavior using data analytics techniques.  
It involves data loading, cleaning, exploratory data analysis (EDA), SQL querying, and building interactive dashboards to generate meaningful business insights.

---

## 📂 Dataset
The dataset contains customer transaction data, including:
- Customer details  
- Product information  
- Purchase amount  
- Date and time of transactions  

---

## 🛠️ Tools & Technologies
- Python – Data loading, cleaning, and EDA  
- PostgreSQL / MySQL / SQL Server – Data querying and analysis  
- Power BI – Dashboard creation and visualization  
- Gamma – Presentation (PPT) creation  

---

## 🔄 Project Steps

### 1. Data Loading
- Imported dataset using Python  
- Checked structure and data types  

### 2. Data Cleaning
- Handled missing values  
- Removed duplicates  
- Standardized formats  

### 3. Exploratory Data Analysis (EDA)
- Analyzed customer spending patterns  
- Identified top products and categories  
- Observed trends over time  

### 4. SQL Analysis
- Wrote queries to extract insights  
- Used aggregations (SUM, AVG, COUNT)  

### 5. Dashboard Creation
- Built interactive dashboards in Power BI  
- Visualized key metrics and trends  

### 6. Reporting
- Created presentation summarizing findings  
- Highlighted key insights and recommendations  

---

## 📊 Dashboard
The Power BI dashboard includes:
- Sales trends over time  
- Top-selling products  
- Customer segmentation  
- Revenue insights  

---

## 📈 Results
- Identified customer purchasing patterns  
- Found high-performing products and categories  
- Generated insights to improve business strategies
